# Personal Portfolio Website

This is my personal portfolio website built to showcase my works, skills, and journey as a Computer Engineering student, Software Developer, and QA Engineer.

## 🚀 Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (React)
- **Styling**: [Chakra UI](https://chakra-ui.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/) & [LottieFiles](https://lottiefiles.com/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Deployment**: Vercel
